# Save English Vocabulary to Notion

英語のwebページを読んでいるときや勉強しているときに出てきた知らない英単語をNotionに保存するChrome拡張機能です。

## 機能

- 英単語をNotionデータベースに保存
- テキスト選択からの自動入力
- 文脈（コンテキスト）の保存
- ソースURL（どのページで見つけたか）の記録
- 右クリックメニューからの素早い保存
- 日付の自動記録

## セットアップ

### 1. Notionデータベースの作成

Notionで新しいデータベースを作成し、以下のプロパティを追加してください：

| プロパティ名 | タイプ | 説明 |
|------------|--------|------|
| 単語 | タイトル | 保存する英単語 |
| 文脈 | リッチテキスト | 単語が使われている文章や文脈 |
| URL | URL | 単語を見つけたページのURL |
| 日付 | 日付 | 保存した日付 |

### 2. Notion APIキーの取得

1. [Notion Integrations](https://www.notion.so/my-integrations)にアクセス
2. 「New integration」をクリック
3. インテグレーション名を入力（例：「Vocabulary Saver」）
4. 作成したインテグレーションの「Internal Integration Token」をコピー

### 3. データベースIDの取得

1. Notionで作成したデータベースページを開く
2. URLをコピー
3. URLから以下の形式でデータベースIDを取得：
   ```
   https://www.notion.so/{workspace_name}/{database_id}?v={view_id}
   ```
   `database_id`の部分（32文字の英数字）がデータベースIDです

### 4. データベースへのアクセス権限の付与

1. Notionデータベースページの右上の「...」メニューをクリック
2. 「Add connections」を選択
3. 作成したインテグレーションを選択して追加

### 5. 拡張機能の設定

1. 拡張機能のポップアップを開く
2. 「設定」タブをクリック
3. Notion API KeyとDatabase IDを入力
4. 「設定を保存」をクリック

## 使い方

### 方法1: ポップアップから保存

1. 保存したい英単語を選択（オプション）
2. 拡張機能のアイコンをクリック
3. 単語と文脈（オプション）を入力
4. 「Notionに保存」をクリック

### 方法2: 右クリックメニューから保存

1. 保存したい英単語またはフレーズを選択
2. 右クリック
3. 「Save "{選択したテキスト}" to Notion」を選択

## 注意事項

- Notion APIキーとデータベースIDは拡張機能内に安全に保存されます
- 拡張機能はインターネット接続が必要です
- Notion APIの利用制限に注意してください

## トラブルシューティング

### 保存できない場合

1. API KeyとDatabase IDが正しく設定されているか確認
2. データベースにインテグレーションのアクセス権限が付与されているか確認
3. データベースのプロパティ名が正確に一致しているか確認（「単語」「文脈」「URL」「日付」）
4. Notionのサービスステータスを確認

## プライバシー

- この拡張機能はローカルストレージにのみデータを保存します
- データはNotionにのみ送信されます
- 第三者にデータを送信することはありません
