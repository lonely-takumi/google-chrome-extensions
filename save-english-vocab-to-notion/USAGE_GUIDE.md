# 使用ガイド / Usage Guide

## 拡張機能の概要

この拡張機能は、英語学習中に出会った知らない単語を素早くNotionに保存するためのツールです。

## セットアップ（初回のみ）

### ステップ 1: 拡張機能をインストール

1. Chrome拡張機能として読み込む
2. 拡張機能のアイコンがツールバーに表示される

### ステップ 2: Notionを設定

詳細は `INSTALL.md` を参照してください。

1. Notionでデータベースを作成
2. 必要なプロパティを追加（単語、文脈、URL、日付）
3. Notion APIキーを取得
4. データベースIDを取得

### ステップ 3: 拡張機能を設定

1. 拡張機能のアイコンをクリック
2. 「設定」タブを開く
3. APIキーとデータベースIDを入力
4. 「設定を保存」をクリック

## 使い方

### 方法1: ポップアップから保存（推奨：初回や確認したい時）

#### ステップ1: テキストを選択

```
Webページで保存したい英単語やフレーズを選択します。

例: "ephemeral"
```

#### ステップ2: 拡張機能を開く

```
ツールバーの拡張機能アイコンをクリックします。
```

#### ステップ3: 内容を確認・編集

ポップアップが開き、以下が表示されます：

```
┌─────────────────────────────────────┐
│ Save Vocabulary to Notion           │
├─────────────────────────────────────┤
│ [単語を保存] [設定]                 │
├─────────────────────────────────────┤
│                                     │
│ 単語:                               │
│ [ephemeral                        ] │
│                                     │
│ 文脈 (オプション):                  │
│ [The ephemeral nature of cherry   ] │
│ [blossoms...                      ] │
│ [                                 ] │
│                                     │
│ ソースURL:                          │
│ https://example.com/article         │
│                                     │
│ [ Notionに保存 ]                    │
│                                     │
│ 💡 ヒント: テキストを選択してから   │
│    拡張機能を開くと、自動的に入力   │
│    されます                         │
└─────────────────────────────────────┘
```

#### ステップ4: 保存

「Notionに保存」ボタンをクリックします。

成功すると以下のメッセージが表示されます：
```
✓ Notionに保存しました！
```

### 方法2: 右クリックメニューから保存（推奨：素早く保存したい時）

#### ステップ1: テキストを選択

```
保存したい英単語やフレーズを選択します。

例: "serendipity"
```

#### ステップ2: 右クリックメニューから保存

```
選択したテキストを右クリック
↓
"Save "serendipity" to Notion" を選択
↓
即座に保存！
```

#### ステップ3: 通知を確認

画面右上に通知が表示されます：
```
┌─────────────────────────────────┐
│ Saved to Notion                 │
│ "serendipity" を保存しました    │
└─────────────────────────────────┘
```

## 実際の使用例

### 例1: 短い単語を保存

**シナリオ**: ニュース記事を読んでいて "ubiquitous" という単語に出会った

```
1. "ubiquitous" を選択
2. 右クリック → "Save "ubiquitous" to Notion"
3. 完了！
```

**Notionに保存される内容**:
```
単語: ubiquitous
文脈: (空)
URL: https://news.example.com/article-123
日付: 2025-11-08
```

### 例2: 文脈付きで保存

**シナリオ**: 長い文章を選択して文脈と一緒に保存したい

```
1. "The ubiquitous presence of technology has changed our lives"
   を選択
2. 拡張機能アイコンをクリック
3. ポップアップで確認:
   - 文脈: "The ubiquitous presence of technology has changed our lives"
4. 「Notionに保存」をクリック
```

**Notionに保存される内容**:
```
単語: The ubiquitous presence of technology has...
文脈: The ubiquitous presence of technology has changed our lives
URL: https://blog.example.com/tech-article
日付: 2025-11-08
```

### 例3: 単語を追加・編集してから保存

**シナリオ**: 自動入力された内容を修正してから保存したい

```
1. テキストを選択（または選択しない）
2. 拡張機能アイコンをクリック
3. 単語フィールドに "pragmatic" と入力
4. 文脈フィールドに "Being pragmatic means dealing with things
   sensibly" と入力
5. 「Notionに保存」をクリック
```

## ヒントとコツ

### 💡 効率的な使い方

1. **読書中は右クリックメニューを使う**
   - 流れを中断せずに素早く保存できます

2. **重要な単語はポップアップから保存**
   - 文脈を追加できるので、後で復習しやすくなります

3. **複数の単語を連続して保存**
   - 右クリックメニューを使えば、何度もポップアップを開く必要がありません

### 📚 学習への活用

1. **毎日の目標を設定**
   - 1日に5〜10個の新しい単語を保存

2. **定期的に復習**
   - Notionで保存した単語を日付順に確認

3. **文脈を重視**
   - 単語だけでなく、使われている文章も保存することで記憶に定着しやすくなります

## トラブルシューティング

### 問題: 保存できない

**確認事項**:
- [ ] APIキーが正しく設定されているか
- [ ] データベースIDが正しく設定されているか
- [ ] インテグレーションがデータベースに接続されているか
- [ ] インターネット接続があるか

### 問題: テキストが自動入力されない

**解決方法**:
1. ページをリロード
2. テキストを選択し直す
3. 拡張機能を再度開く

### 問題: 右クリックメニューが表示されない

**解決方法**:
1. 拡張機能が有効になっているか確認
2. ページをリロード
3. Chromeを再起動

## キーボードショートカット

現在、デフォルトのキーボードショートカットはありませんが、以下の方法で設定できます：

1. `chrome://extensions/shortcuts` を開く
2. "Save English Vocabulary to Notion" を探す
3. ショートカットを設定（例: Ctrl+Shift+V）

## Notionでの活用

### ビューの作成

Notionで以下のようなビューを作成すると便利です：

1. **最近保存した単語**
   - 日付でソート（降順）

2. **復習が必要な単語**
   - フィルター: 日付が7日以上前

3. **カテゴリ別**
   - タグでグループ化（将来の機能）

### データベーステンプレート

```
| 単語      | 文脈                           | URL                  | 日付       |
|-----------|--------------------------------|---------------------|------------|
| ephemeral | The ephemeral nature of...    | https://example.com | 2025-11-08 |
| pragmatic | Being pragmatic means...       | https://blog.com    | 2025-11-08 |
```

- ✅ ソースURLを自動記録
- ✅ 後でNotionで簡単に復習

英語学習の効率が大幅に向上します！
